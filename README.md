# cinema
POC0 – Mini Herzog cinema

FirstKnonwInterplanetaryWernerHerzogFilmArchive

This POC is a single-page, browser-based WebTorrent viewer for the “Werner Herzog Collection”
infohash. It:
- auto-loads the Herzog collection into a mini library + player
- exposes the collection infohash and a simple WebTorrent client panel
- is styled to stay minimal, calm, and readable
